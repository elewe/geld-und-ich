# Pages

Dieser Ordner kann für zusätzliche Seitenkomponenten oder Utilities verwendet werden.

Hinweis: Da dieses Projekt Next.js App Router verwendet, befinden sich die Hauptseiten im `app/` Verzeichnis.
Dieser Ordner kann für:
- Geteilte Seitenkomponenten
- Seiten-Utilities
- Oder andere projekt-spezifische Seiten-Dateien verwendet werden.
